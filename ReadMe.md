# Dolphin Pong

A fun twist on the classic Pong game, featuring a dolphin as the ball and roses as paddles!

## How to Play

1. Make sure you have Python installed on your computer
2. Install Pygame by running: `pip install pygame`
3. Download all files from this repository
4. Run `python pong.py` in your terminal/command prompt

### Controls
- Up Arrow: Move paddle up
- Down Arrow: Move paddle down

## Requirements
- Python 3.x
- Pygame library
